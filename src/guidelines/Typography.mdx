Wir verwenden SCSS-Mixins, um konsistente Typografiestile anzuwenden. Nachfolgend finden Sie eine Übersicht der verfügbaren Mixins und deren Verwendungszweck.

<div className="typo-header">Display Stil:</div>

Das Mixin `L-Display-Style` wird für prominente Überschriften oder Titel verwendet. Es unterstützt die folgenden Varianten:
- Regular (Standard)
- Semibold
- Bold

```scss
@mixin L-Display-Style($variant: null) {
  font-family: Lato, sans-serif;
  font-size: theme.token(font-size-xxl);
  font-style: normal;
  line-height: normal;
  letter-spacing: normal;

  @if $variant == '-semibold' {
    font-weight: theme.token(font-weight-semibold);
  } @else if $variant == '-bold' {
    font-weight: theme.token(font-weight-bold);
  } @else {
    font-weight: theme.token(font-weight-regular);
  }
}
